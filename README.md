# Insight Mídia - Análise de Desempenho de Campanhas (SQL)

Este projeto simula o trabalho de um analista de dados em uma agência de publicidade, utilizando SQL para analisar campanhas do Google Ads.

## Objetivo
Explorar e analisar dados de campanhas publicitárias, calculando métricas de desempenho e insights de ROI.

## Estrutura das Tabelas
- **campanhas:** informações gerais das campanhas (cliente, orçamento, período)
- **resultados:** métricas de performance (impressões, cliques, conversões, custo)

## Consultas Desenvolvidas
1. Top 5 campanhas com mais conversões  
2. Cálculo de CTR e CPC  
3. Cálculo de ROI estimado  
4. Campanhas com CTR abaixo da média geral  

## Insights
- A campanha **Black Friday (TechNova)** teve o maior número de conversões e ROI positivo.  
- Algumas campanhas estão com **CTR abaixo da média**, indicando necessidade de otimização criativa.  

## Tecnologias Utilizadas
- **MySQL** → modelagem e consultas SQL  
- **GitHub** → versionamento e portfólio
